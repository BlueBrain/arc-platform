{% trans 'Request status' %}
{% trans 'Open' %}

A request is open as long as the supplier is looking for the equipment, reagent, consumable, know-how or personnel. While a request is open, it is possible for the moderators and validators to find supplies and to attribute them to the request.

{% trans 'Closed' %}

A request can be closed when the need has been fulfilled. This action is set manually by the moderator or validator. A request cannot be closed as long as there is at least one attributed supply with the status “on going attribution”.

{% trans 'Archived' %}

A request can be archived only when no supply is attributed to the request. A archived supply may indicate that the request is a duplicate, or that the request has been provided in another way.

{% trans 'Supply status' %}
{% trans 'Available' %}

A request is available as long as the supplier offers the equipment, reagent, consumable, know-how or personnel. While a supply is available, it is possible for the moderators and validators to attribute it to a request.

{% trans 'On going attribution' %}

A request is on going attribution when the supply is attributed to a request. A validator need to validate or to dismiss manually the request attribution.

{% trans 'Attributed' %}

A request is attributed when a supply has been validated for an attributed request.

{% trans 'Closed' %}

A supply can be closed when the supply has been attributed. This action is set manually by the moderator or validator. A supply cannot be closed when its status is “on going attribution” nor when when there is no attributed request.

{% trans 'Archived' %}

A supply can be archived only when no request is attributed to the supply. A archived supply may indicate that the supply is a duplicate, or that the supply has been offered in some other way.
